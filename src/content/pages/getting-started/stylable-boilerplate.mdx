Stylable offers a [boilerplate utility](https://www.npmjs.com/package/create-stylable-app) that can be used to bootstrap new projects from a list of templates.

## Create a new project

```bash
npx create-stylable-app <project-name>
```

You will get a choice of the following templates:

- `ts-react-webpack` (default) - based on React, TypeScript and webpack
- `ts-react-rollup` - based on React, TypeScript and Rollup

## Install dependencies

From there you are going to want change into your `project-name` directory and install Shelley.

```bash
npm i @actionishope/shelley
```
